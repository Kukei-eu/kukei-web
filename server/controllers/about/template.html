What is it

Welcome to a curated search engine aimed at web developers.

It's getting matured, so much that I started using it at work, but still very much in progress with growing index of:

  • {{blogPages}} blogs pages
  • {{docsPages}} docs pages
  • {{magazinesPages}} magazines pages

It's created and maintained by Szymon Nowicki, you can also check all the source code on GitHub

You can always contact me on Mastodon or hey@nowicki.io if you have any suggestions or feedback. I'll read all of it.

Privacy policy

This website is hosted on my living room PC. Backend, so a search index, crawlers, and companion mongodb are hosted in my living room as well.

It doesn't use any extensive tracking, only Plausible.io powered, backend proxied tracking to further anonymize visitors IP.

Whole networking model is protected by CloudFlare so technically CloudFlare is able to see what you're doing.

Additionally, I log all queries in a database, for me, to periodically look at them so I can improve quality. Once I look at them I purge this database table completetly.
